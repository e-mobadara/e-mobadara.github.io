Installation

Step One

Add the JitPack Repository to your build path

In your builde.gradle (Project: project_name)

                                        
allprojects {
    repositories {
        ...
        maven { url 'https://jitpack.io' }
    }
}

Step Two

Add The Dependency

This time in your builde.gradle (Module:app)

                                            
dependencies {
    ...
    implementation 'com.github.e-mobadara:moblib:v1.2.1'
}


It is good to have the Repository as watched to get notified about new Releases

That's It! Now You are Good to Go

Philosophy

The most important class in this library is AndroidGame.
It extends Activity and implements Game.
AndroidGame is the One responsible for what screen is showing in the moment.
Each Screen is responsible for the sprites that are showing.
Screen have an ArrayList of sprites, and in order for a sprite to be shown it has to be added to this List.

The current Image says it all
